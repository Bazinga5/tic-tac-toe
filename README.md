# Tic

A tic tac toe implementation.

Test reports are located at: [bazinga5.github.io/tic-tac-toe](http://bazinga5.github.io/tic-tac-toe/)

## Usage:

Build:
```sh
$ ./gradlew build
```

Test:
```sh
$ ./gradlew test
```

Run server:
```sh
$ ./gradlew run
```
